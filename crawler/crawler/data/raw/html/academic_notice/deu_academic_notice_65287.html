2021-2학기 혁신공유대학 교과목 수강 안내

  • 작성일:2021-10-21
  • 작성자:학사지원팀
  • 조회수:2596
첨부파일 학습자 매뉴얼.pdf

1. 수업이수 기간 : 2021.10.25.(월) ~ 2021.12.17.(금) (*집중이수제)

일정

학습 기간

비고

11.15.(월) ~ 11.19.(금)

중간시험 기간

강좌별 공지 확인

12.13.(월) ~ 12.17.(금)

기말시험 기간

강좌별 공지 확인

* 주차별 출석 인정 학습 기간은 강좌별 상이하므로 반드시 확인하여 수강 바람

 

2. 수업참여 방법 : 바이오헬스 COSS 플래닛 http://lms.biocoss.ac.kr/ (* DOOR시스템에서는 강의 수강 X)

- ID : 학교코드 C(대문자)+학번 (예 : 학번이 20211234일 경우 C20211234)

- PW: 학교코드 C(대문자)+학번 (초기 비밀번호는 ID와 동일한 C20211234로, 이후 PW 변경 가능)

※ 첨부파일 ‘학습자 매뉴얼’ 참고

 

3. 교과목 성적평가방법

- 5D 진로설계 과목은 (P/F)로 성적산출, 이외의 과목은 절대평가 실시

- 평가 세부항목은 교과목별 강의계획서 참고 (https://biohealthuniversity.oopy.io/)

 

4. 바이오헬스 COSS 플래닛 시스템 문의

- 바이오헬스 COSS 플래닛 http://lms.biocoss.ac.kr/ 로그인>사이버캠퍼스>커뮤니티>고객지원 게시판

- 문의를 남길 때는 이슈가 발생한 교과목, 담당교수, 어떤 이슈가 발생했는지 구체적으로 남겨야

빠른 확인 및 피드백 가능 (※ 교과목 관련 문의 피드백 시간 : 평일 09시~18시까지 / 주말, 공휴일 제외)

- 과목정보/ 재생이 불가한 차시명/ 보다 구체적인 현상/ 어떤 브라우저를 사용했는지를 기재하면

빠른 피드백 가능

(예 : ‘바이오헬스 인체의 신비’ 홍길동 교수님의 강의에서 1주차 1차시 영상이 재생 되다가 3분

정도에 갑자기 멈추는 현상이 발생했습니다. 강의영상은 크롬(Chrome)으로 연결했습니다.)

 

5. 관련문의

- 바이오헬스 혁신공유대학사업단 행정지원실 T. 051-890-3011~3, 3015
