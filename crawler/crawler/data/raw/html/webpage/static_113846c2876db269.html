IT서비스

무선인터넷 Cloud 서비스(이메일) 헤이영캠퍼스 기타

사용안내

U-세대에 걸맞는 유비쿼터스 교육환경을 실현하여
A.I(캠퍼스의 인공지능화) Virtual(가상강의)등을 실현 할수 있는 최적화된 새로운 U-Campus 문화를 확립합니다.

무선랜 규격

  • 교내 무선랜(Wi-Fi)을 사용하기 위한 보안대책으로 무선랜 보안 프로토콜 WPA2(Wi-Fi Protected Access 2)를 지원하도록 정책을 설정하였음.
    ※ 관련 근거 : 국가정보원 보안지침(新정보통신 기술활용에 따른 보안대책 강구)에 의거 무선구간 보호를 위해 국가정보원장이 안전성을 검증한 암호시스템을 사용하고 단말기에 대한 인증 강화.
  • WPA2를 지원하는 무선랜카드 규격 : 802.11g(대부분 내장형), 802.11n
  • 802.11b나 802.11a는 대부분이 WPA2 방식을 지원하지 않으므로 교내무선랜을 사용하기 위해서는 이에 맞는 보안 프로토콜을 지원하는 소프트웨어를 설치하여야만 사용이 가능 함.
  • PDA 무선 보안 인증을 위해서는 아래에 링크 되어 있는 인증서 파일을 다운로드 받아 PDA에 설치하여야 함. 아래에 있는 PDA 무선랜 사용자 설명서를 참조.
    ※ 소프트웨어 : 무선랜카드 Vender에서 직접 지원하는 소프트웨어를 말함
PDA인증서 다운로드

사용방법

교내 와이파이는 특별열람증 및 외부인에게 제공되지 않습니다.

서비스 구역, 서비스 정책, 서비스 내용, 서비스 사용방법으로 구성
서비스 구역 동의대학교 가야·양정캠퍼스 전체
서비스 정책 교직원, 시간강사, 학생 : 유선랜 사용과 동일한 범위에서 사용 가능(전자문서, 외부인터넷, 학생지원시스템 등 접근가능)
※ 교직원 : 재직중인 사람 / 시간강사 : 당해연도 강의를 하는 사람 / 학생 : 재(휴)학생, 졸업생
서비스 내용
  1. 무료로 즐기는 자유로운 무선인터넷
  2. 학사행정 서비스 (수강신청, 학적 및 성적관리 등)
  3. 학내 Wi-Fi 무료서비스
서비스 사용방법
  1. 1단계 : 무선네트워크 환경 설정(최초 1회)넷
  2. 2단계 : 사용 가능한 무선네트워크(Wi-Fi))
  3. 3단계 : 사용자 인증 및 로그인 후 사용

자세한 사용방법은 하단 붙임 참조

기타사항

교내에서 개별적으로 설치하여 사용하고 있는 무선랜 AP는 인증을 받지 않을 경우 보안에 매우 취약하며, 또한 이상 트래픽을 발생시켜 전체사용자에게 불편을 초래할 수 있으므로 반드시 전산정보원에 신고하여 인증시스템에 추가로 등록하여야 함.
